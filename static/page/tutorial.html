Tutorial

  1. Masukkan gambar yang ingin dicari pada tombol "unggah gambar"

  2. Jika unggah gambar berhasil, maka akan tampil gambar pada bagian kiri

  3. Masukkan dataset yang ingin digunakan pada tombol "Unggah folder dataset"

  4. Tentukan jenis pencarian (Warna/Tekstrur)

  5. Tekan tombol submit

  6. Tunggu beberapa saat, hasilnya akan keluar di bagian bawah
